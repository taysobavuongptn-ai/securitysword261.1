SecuritySword

Lightweight Manual Malware Scanner for Windows

About SecuritySword

SecuritySword is a lightweight Windows desktop application designed to scan folders and removable drives for known malicious files using SHA-256 hash comparison.

The software performs scans only when explicitly initiated by the user. It does not run as a background service and does not modify system security settings.

Key Features

  • Manual scan only (no background service)
  • Folder and removable drive scanning
  • SHA-256 hash-based detection
  • Quarantine for detected files
  • Clear graphical user interface

System Requirements

  • Windows 10 / 11 (64-bit)
  • No administrator privileges required
